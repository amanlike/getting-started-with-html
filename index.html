Important Concepts

Basics of the web

The web is made up of computers that are all linked over the internet. You use a browser such as Internet Explorer, Firefox or Chrome to send a request over the internet using a protocol called HTTP which stands for Hypertext Transfer Protocol. When type in an address such as amazon.com your computer sends a request to the hosting server which responds back to your browser with the specific page you requested which also can contain links to other pages on the same site or to external sites.

HTML

HTML or Hypertext Markup Language is the language used to display content in your browser. It is made of text content as in what you see, markup which is how it looks and references to other documents such as images and videos and also links to other web pages which can be local or external.

Garbage in, Garbage out

Computers generally follow instructions on how and what task to complete. If these instructions contain a single character typo then computers are unforgiving and will most of the time fail in the requested task. This is why it is of high importance to have accurate data entered initially.

The make up of a basic webpage

A basic webpage is made up tags and elements which define how the webpage is laid out. There are numerous tags which can display the text in a multiple of formats such bold, italic, underline etc. A collection of tags can be known as an element where you can inline or block. Inline means to span text and block means to put an area round the text like an invisible block around the text
